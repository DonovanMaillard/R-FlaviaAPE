Installation
------------

Installer le packages "remotes" pour pouvoir récupérer le package RFlaviaAPE

.. code:: console

	$ install.packages("remotes")
	$ library(remotes)

Puis récupérer le package RFlaviaAPE

.. code:: console

	$ install_github("DonovanMaillard/RFlaviaAPE", force=TRUE)
	$ library(RFlaviaAPE)
